\textbf{Logical Negation}
Please state the negation of the following statements.
\begin{enumerate}
	\item It rains.
	\item It rains and it is cold outside.
	\item $A \land B$ 
	(\textit{Hint: }De Morgan)
	\item $x = 3 $ or $x = 1$
	\item $A \lor B$
	\item If it rains the street gets wet.
	\item $A \Rightarrow B$ 
\end{enumerate}
