\section{Symmetric Matrices}
	Which of the following three matrices is symmetric?
\begin{enumerate}
	\item$A=\begin{pmatrix}
	0&0&1&2\\
	0&0&3&4\\
	1&2&0&0\\
	3&4&0&0
	\end{pmatrix}$
	\item $B=\begin{pmatrix}
	1&2&0&0\\
	3&4&0&0\\
	0&0&4&2\\
	0&0&3&1
	\end{pmatrix}$
	\item $C=\begin{pmatrix}
	0&0&1&2\\
	0&0&3&4\\
	1&3&0&0\\
	2&4&0&0
	\end{pmatrix}$
\end{enumerate}
